## 1. 会計基準の選択

### 1.1 適用可能な会計フレームワークの全体像

日本のスタートアップは、規模・上場区分・将来計画（IPO 国 / M&A 戦略）に応じて、以下の会計フレームワークから選択する必要がある。フレームワーク選択は税務との接続、監査コスト、海外投資家との対話、IPO ロードマップに直接影響する。

| フレームワーク | 想定適用主体 | 強制 / 任意 | 主な設定主体 |
|---|---|---|---|
| 中小企業会計要領 | 規模の小さい中小企業（株主が極めて少数） | 任意 | 中小企業庁・日本商工会議所等 |
| 中小企業会計指針 | 一定規模の中小企業（会計参与設置会社等） | 任意 | 日本税理士会連合会等 |
| 一般的な会計基準（日本基準 J-GAAP） | 上場会社・上場準備会社・大会社 | 強制（金商法・会社法 大会社） | ASBJ（企業会計基準委員会） |
| IFRS（国際財務報告基準） | 任意適用会社 | 任意 | IASB |
| 修正国際基準（JMIS） | 任意適用 | 任意 | ASBJ |
| US-GAAP | 米国上場目的等 | 限定 | FASB |

スタートアップの段階別では、創業期は中小企業会計要領 → シリーズ A 以降は中小企業会計指針 → IPO 直前 N-2 期から一般的な会計基準（連結対応）に移行するのが定石である。

### 1.2 中小企業会計要領 vs 中小企業会計指針

「中小指針」は 2005 年制定、「中小要領」は 2012 年制定で、対象規模・会計処理の精緻さに違いがある。

| 項目 | 中小企業会計要領 | 中小企業会計指針 |
|---|---|---|
| 対象 | 小規模・株主極少 | 一定規模・会計参与設置会社等 |
| 税効果会計 | 不要 | 必要（重要性のあるもの） |
| 組織再編会計 | 限定的 | 必要 |
| 退職給付 | 簡便法 | 簡便法可（ただし注記要件あり） |
| 金融商品 | 簡便法 | 時価評価原則 |
| ストックオプション | 任意 | 公正な評価額で費用計上 |

**モデリング上の影響**

- **損益**：要領を採用すると SO 費用計上が省略可能で、見かけ上の人件費は低く出る。指針 / 一般基準に移行すると過去の SO についても会計処理を再構築する必要があり、IPO 準備時に "黒字 → 赤字転落" するケースがある。
- **キャッシュフロー**：会計処理選択は CF に影響しないが、税効果計上の有無で繰延税金資産（DTA）が動く。
- **希薄化**：直接影響なし。ただし IPO 準備で SO 費用化が顕在化したとき、評価額算定に用いるブラックショールズ・モデルのインプット（無リスク利子率 / ボラティリティ / 残存期間）を将来期間に外挿する必要がある。
- **勘定科目**：株式報酬費用 / 新株予約権（純資産の部）。

### 1.3 一般的な会計基準（J-GAAP）における主要論点

#### 1.3.1 収益認識基準（企業会計基準第 29 号）

**適用時期**：2021 年 4 月 1 日以後開始する連結会計年度・事業年度から強制適用（中小指針適用会社は適用免除可）。

**5 ステップ**

1. 顧客との契約の識別
2. 履行義務の識別
3. 取引価格の算定
4. 取引価格の履行義務への配分
5. 履行義務の充足時 / 充足につれての収益認識

**SaaS / サブスクリプションの典型処理**

- 月額利用料：一定期間にわたり充足される履行義務 → 期間按分
- 初期設定費（セットアップ・フィー）：契約期間にわたり繰延（独立販売価格テストを通る場合は別個の履行義務）
- 多年契約の前受金：契約負債として計上、月次取り崩し
- カスタマイズ開発：ソフトウェア完成 / 検収時点で収益計上、または進捗度で配分

**B2B 受託開発・SI 案件**

- 一時点認識（検収基準）または期間にわたる認識（進捗度ベース）を契約毎に判定
- 工事進行基準は廃止され、進捗度合理的測定可能性が要件

**モデリング上の影響**

- **損益**：MRR / ARR の P/L 反映は契約負債（=Deferred Revenue）の取り崩し。Booking ≠ Revenue であることを明示してモデル化。
- **キャッシュフロー**：年契約前払いの場合、初月に CF 12 カ月分を受領するが収益は 1 カ月分。CCC（cash conversion cycle）にプラスに働く。
- **希薄化**：直接影響なし。
- **勘定科目**：契約負債（流動 / 固定で区分）、契約資産、返金負債。

#### 1.3.2 リース会計基準（企業会計基準第 34 号）

**適用時期**：2027 年 4 月 1 日以後開始する事業年度から強制適用、2025 年 4 月 1 日から早期適用可能。

**主な変更**

- 借手側：オペレーティング・リースを含むすべてのリースで使用権資産・リース負債を計上（IFRS 16 とほぼ同等）。
- ファイナンス・リースとオペレーティング・リースの区分は借手側で消滅。
- 短期リース（12 カ月以内）・少額リース（個別少額）には簡便処理あり。

**スタートアップで影響が大きい契約**

- オフィス賃貸借（普通借家・定期借家）
- データセンター / コロケーション
- 営業車両・配送車両
- SaaS の長期コミット（リース該当性の判定が必要）

**モデリング上の影響**

- **損益**：賃料定額計上 → 減価償却費 + 支払利息に分解。期初は支払利息が大きく、利益が下振れ（front-loaded）。
- **キャッシュフロー**：CF 総額は不変だが、営業 CF（利息支払を金融区分する場合）が改善し、財務 CF が悪化。EBITDA が見かけ上改善（賃料 → 減価償却 + 利息に切替）。
- **希薄化**：影響なし。
- **勘定科目**：使用権資産（無形固定 or 有形固定）、リース負債（流動 / 固定）。

> **モデリング Tips**：早期適用する場合、過年度比較数値の遡及修正（簡便法 or 完全遡及）を選択。VC との対話で EBITDA を強調するスタートアップは、新基準で EBITDA が嵩上げされる点を説明し、業績比較の連続性に注意。

#### 1.3.3 のれん（企業会計基準第 21 号「企業結合に関する会計基準」）

**日本基準**：のれんは原則 20 年以内のその効果が及ぶ期間で規則的償却 + 減損兆候時に減損テスト。負ののれんは発生事業年度の特別利益。

**IFRS**：のれんは非償却、毎期減損テスト（少なくとも年 1 回）。負ののれんは発生年度の利益（IFRS 3）。

| 項目 | 日本基準 | IFRS | US-GAAP |
|---|---|---|---|
| のれん償却 | 20 年以内で規則的償却 | 非償却 | 非償却（2014 年改正以降は私企業選択可） |
| 減損テスト | 兆候判定 → 認識 → 測定の 3 段階 | 毎期 1 回以上 | 毎期 1 回以上（2 段階 → 1 段階に簡素化） |
| 戻入 | 不可 | 不可（のれん）/ 可（その他資産） | 不可 |

**モデリング上の影響**

- **損益**：M&A を多用するスタートアップ（ロールアップ型）では、日本基準のほうが利益が低く出る。20 年償却なら買収のれん 20 億円 → 年 1 億円の償却費。
- **キャッシュフロー**：のれん償却は非現金費用なので CF 直接影響なし。EBITDA → EBITA（のれん償却前利益）でブリッジ可。
- **希薄化**：株式対価 M&A の場合は希薄化発生（章 10 参照）。

#### 1.3.4 ストックオプション会計（企業会計基準第 8 号）

**原則**：付与日の公正な評価額（典型的にはブラックショールズ式）を、対象勤務期間にわたり費用計上。

**会計処理**

```
（付与日 〜 権利確定日まで毎期）
株式報酬費用 / 新株予約権（純資産の部）
```

**ブラックショールズ式の主要インプット（日本実務）**

- 株価：未上場株は直近ラウンドの株価 / DCF / 類似会社比較で算出
- 行使価額
- 残存期間：付与から権利行使までの予想期間
- ボラティリティ：類似上場企業のヒストリカル・ボラティリティ
- 無リスク利子率：日本国債利回り
- 配当利回り：通常 0%

**モデリング上の影響**

- **損益**：株式報酬費用（販管費）として計上。損金不算入が原則のため、会計上の費用と税務上の損金にズレ → 永久差異（税効果なし）が生じやすい。
- **キャッシュフロー**：非現金費用、CF 直接影響なし。
- **希薄化**：付与済 SO は dilution の母集団に算入（fully diluted）。
- **勘定科目**：株式報酬費用（販管費）、新株予約権（純資産の部）。

#### 1.3.5 R&D（研究開発費等に係る会計基準）

**原則**：日本基準では研究開発費は発生時に費用処理（資産計上不可）。例外的にソフトウェアの一部（自社利用 / 市場販売目的）が条件付きで資産化可能。

**ソフトウェアの会計処理**

| 用途区分 | 研究段階 | 開発段階（将来収益見込あり） | 製品マスター完成後 |
|---|---|---|---|
| 自社利用 | 費用 | 費用 | 資産（5 年以内償却） |
| 市場販売目的 | 費用 | 費用 | 資産（製品マスター 3 年以内 + 著しく減価する場合は短縮） |
| 受託開発 | 費用 | 費用 | 仕掛品 |

**IFRS との比較**：IFRS は研究フェーズは費用、開発フェーズで一定要件（実行可能性・将来便益・意図 etc, IAS 38）を満たせば資産化が強制。ディープテック / バイオでは IFRS の方が初期赤字が小さく見える。

**モデリング上の影響**

- **損益**：日本基準 R&D は full expense → スタートアップは初期赤字が大きくなる。
- **キャッシュフロー**：研究開発税額控除（章 2 参照）の効果は、納税額が出始めてから顕在化。
- **希薄化**：影響なし。
- **勘定科目**：研究開発費（販管費）、ソフトウェア（無形固定資産）、ソフトウェア仮勘定。

### 1.4 IFRS 任意適用

#### 1.4.1 適用要件

東証は連結財務諸表の任意適用を認めている。IFRS 強制適用は導入されていない（2026 年現在、引き続き任意適用）。

**任意適用の主な要件**（金商法・連結財務諸表規則）

- 上場会社または上場予定会社
- 国際的な財務活動・事業活動
- 適切な体制（IFRS 知見ある経理体制 / 内部統制）

#### 1.4.2 IFRS と J-GAAP の主要差分まとめ

| 項目 | J-GAAP | IFRS |
|---|---|---|
| のれん | 20 年以内償却 | 非償却 |
| R&D 開発費 | 費用処理 | 要件充足時資産計上 |
| リース | 2027 年から使用権モデル | 使用権モデル（IFRS 16） |
| 収益認識 | ASBJ 第 29 号（IFRS 15 とほぼ整合） | IFRS 15 |
| 包括利益 | 当期純利益 + その他包括利益 | 同左 |
| 退職給付 | 数理計算上の差異の遅延認識可 | OCI 即時認識 |
| 金融商品分類 | 4 区分（売買目的・満期保有・子会社株式・その他） | SPPI テスト → 償却原価 / FVOCI / FVTPL |
| 再評価 | 不可（一部固定資産の例外を除く） | 有形固定資産・無形資産で選択可 |
| 開発費資産化 | 限定 | 要件充足時強制 |

**モデリング上の影響**

- IFRS 任意適用するとのれん非償却で利益見栄えが改善するが、減損リスクが残る。
- 海外投資家とのコミュニケーションが容易になる。
- 監査コストは初年度大きく増加（差異分析・遡及修正）。

### 1.5 SSBJ（サステナビリティ開示基準）と TCFD

**SSBJ 基準公表**：2025 年 3 月 5 日にサステナビリティ基準委員会（SSBJ）が「サステナビリティ開示基準」を公表（適用基準・一般基準・気候基準）。IFRS S1 / S2 と整合的。

**適用ロードマップ（金融庁案）**

| 適用時期 | 対象 |
|---|---|
| 2027 年 3 月期 | 時価総額 3 兆円以上のプライム上場会社（任意適用は 2026 年 3 月期から） |
| 2028 年 3 月期 | 時価総額 1 兆円以上 |
| 2029 年 3 月期 | 時価総額 5,000 億円以上 |
| 2030 年 3 月期以降 | プライム全企業に拡大検討 |

**TCFD（気候関連財務情報開示タスクフォース）**

- 2021 年改訂コーポレートガバナンス・コードでプライム上場企業に TCFD または同等の枠組みでの開示が要請されている。
- 4 つの開示要素：ガバナンス / 戦略 / リスク管理 / 指標と目標。

**スタートアップへの影響**

- グロース上場時点では強制適用ではないが、プライム移行を狙う場合は早期から温室効果ガス排出量（Scope 1/2/3）の集計体制を準備。
- ESG 評価を投資判断に組み込む VC（特に欧州系・年金系 LP を抱えるファンド）からの開示要請が増加。
- バリュエーションへの影響は限定的だが、調達難易度に影響しうる。

**モデリング上の影響**

- **損益**：開示対応コスト（コンサル・システム）が販管費に。
- **キャッシュフロー**：規模拡大に伴い数百万 〜 数千万円 / 年の追加コスト。
- **希薄化**：直接影響なし。

## 2. 法人税制

### 2.1 法人実効税率の構造（2026 年版）

日本の法人実効税率は、国税（法人税 + 地方法人税 + 防衛特別法人税）と地方税（法人住民税 + 法人事業税 + 特別法人事業税）の組み合わせで決まる。

#### 2.1.1 算定式

```
実効税率 = (法人税率 × (1 + 地方法人税率 + 法人住民税率 + 防衛特別法人税率)
            + 事業税率 + 特別法人事業税率)
           / (1 + 事業税率 + 特別法人事業税率)
```

事業税が損金算入であるため、分母で割り戻す。

#### 2.1.2 主要税率（2026 年 4 月以降の見込み）

| 税目 | 税率 | 根拠 |
|---|---|---|
| 法人税（基本税率） | 23.2% | 法 66 条 |
| 法人税（中小法人軽減税率：所得 800 万円以下） | 15% | 措置法 42 条の 3 の 2 |
| 地方法人税 | 10.3% | 地方法人税法 |
| 防衛特別法人税 | 4%（基準法人税額－500 万円） | 2026 年 4 月以降開始事業年度 |
| 法人住民税（東京都超過課税・標準） | 法人税割 10.4%（標準 7.0% + 超過 3.4%） | 地方税法 |
| 法人事業税（外形標準課税対象法人） | 所得割 1.0% + 付加価値割 1.2% + 資本割 0.5% | 地方税法 |
| 法人事業税（外形対象外） | 所得割 7.0%（軽減税率含む段階課税） | 地方税法 |
| 特別法人事業税 | 事業税所得割の 260% | 特別法人事業税及び特別法人事業譲与税に関する法律 |

#### 2.1.3 実効税率の水準（2026 年版）

| 区分 | 実効税率の目安 | 備考 |
|---|---|---|
| 大企業（資本金 1 億円超、東京都超過税率、防衛特別法人税適用後） | 約 31.52% | 防衛特別法人税導入前は約 30.62% |
| 大企業（外形標準課税対象、防衛特別法人税適用前） | 約 30.62% | デロイト 2026 年税制ニュースレター |
| 中小法人（資本金 1 億円以下、外形非対象） | 約 33 〜 35% | 軽減税率の適用前ベース |
| 中小法人（所得 800 万円以下部分） | 軽減税率 15%（2027 年 3 月 31 日まで） | 措置法 42 条の 3 の 2 |

> **防衛特別法人税の重要ポイント**
>
> - 2026 年 4 月 1 日以後に開始する事業年度から導入。
> - 計算式：(基準法人税額 － 500 万円) × 4%。
> - 中小法人で法人税額が 500 万円以下なら基礎控除によりゼロ。
> - 防衛特別法人税は税額控除（試験研究費税額控除等）後の法人税額をベースに別途計算。

**モデリング上の影響**

- **損益**：税前利益 → 税引後利益への転換係数が異なる。Stage 別（中小 → 大企業移行）でモデルの実効税率を可変にする。
- **キャッシュフロー**：法人税納付は中間納付（事業年度開始から 6 カ月経過後 2 カ月以内）と確定申告（決算後 2 カ月以内、申請で 1 カ月延長可）。
- **希薄化**：影響なし。
- **勘定科目**：法人税、住民税及び事業税 / 法人税等調整額。

### 2.2 中小企業優遇税制

#### 2.2.1 中小法人 / 中小企業者 / 適用除外事業者の区分

| 区分 | 主要要件 | 主な適用税制 |
|---|---|---|
| 中小法人（法 2 条 25 号・措置法 42 の 3 の 2） | 資本金 1 億円以下（大法人の 100% 子会社等を除く） | 軽減税率、欠損金 100% 控除 |
| 中小企業者（措置法 42 の 4 等） | 資本金 1 億円以下 + 大規模法人の 50%/2/3 出資除外 | 投資促進税制、研究開発の中小強化、少額減価償却資産 |
| 適用除外事業者 | 直前 3 年平均所得 15 億円超の中小企業者 | 一部優遇から除外 |

#### 2.2.2 軽減税率と所得分割

中小法人は所得 800 万円以下の部分に対し法人税 15%（措置法 42 の 3 の 2、令和 9 年 3 月 31 日までの間に開始する事業年度）。それ以外は 23.2%。

> 令和 8 年度（2026 年度）税制改正大綱でも軽減税率の 2 年延長が示されている（最新の改正情報を確認のこと）。

#### 2.2.3 交際費課税の特例

- 中小法人：年 800 万円までの定額控除限度額、または接待飲食費の 50% を選択（措置法 61 の 4）。
- 大法人：接待飲食費の 50% のみ。
- 1 人当たり 5,000 円以下の飲食費は交際費から除外可（社外交際費）。1 人 1 万円以下に引き上げる方向（令和 6 年度改正：5,000 円 → 1 万円。スタートアップは令和 6 年 4 月 1 日以後の支出から適用）。

#### 2.2.4 設備投資減税

| 制度 | 対象 | 控除内容 |
|---|---|---|
| 中小企業投資促進税制（措置法 42 の 6） | 機械装置 160 万円以上等 | 取得価額の 30% 特別償却 or 7% 税額控除（資本金 3,000 万円以下） |
| 中小企業経営強化税制（措置法 42 の 12 の 4） | 経営力向上計画認定 | 100% 即時償却 or 10% 税額控除（資本金 3,000 万円以下）/ 7%（同 3,000 万円超） |
| 中小企業デジタル投資促進税制 / DX 投資促進税制 | 認定計画に基づくクラウド・ソフトウェア | 30% 特別償却 or 3% 税額控除（賃上げ要件で 5%） |

**モデリング上の影響**

- **損益**：特別償却は減価償却費を前倒し → 利益を一時的に圧縮、後年戻る（永久差異ではない）。税額控除は永久差異で利益向上。
- **キャッシュフロー**：税額控除は当年度の納税額減 → CF 改善。
- **希薄化**：影響なし。

### 2.3 欠損金繰越控除

- **繰越期間**：10 年（2018 年 4 月 1 日以後開始事業年度の青色申告で生じた欠損金）。
- **控除限度額**：
  - 中小法人：所得金額の 100%
  - 大法人：所得金額の 50%
- 根拠：法 57 条、措置法 66 の 11 の 4 等。

> **コロナ禍特例**：欠損金繰越控除上限の特例（事業適応計画認定法人）は、令和 8 年（2026 年）4 月 1 日以前開始事業年度までを対象。

**モデリング上の影響（極めて重要）**

- **損益**：黒字転換初期は繰越欠損金で課税所得圧縮 → 実効税率が見かけ上低い。モデルでは「税前利益」「課税所得（繰越欠損金控除後）」「法人税等」を分けて計算するべき。
- **キャッシュフロー**：黒字転換期 1 〜 3 期は CF が大きく改善。資本金 1 億円以下なら 100% 控除可、資本金 1 億円超で 50% 控除に切り替わる点に注意。
- **DTA**：繰延税金資産（DTA）の回収可能性判定が IPO 準備期に重要。スケジューリング表で将来課税所得を裏付ける必要。
- **資本金 1 億円超への増資タイミング**は欠損金控除限度に直結。IPO 直前の増資戦略では「資本金 1 億円以下を維持」する設計が一般的。

### 2.4 試験研究費税額控除（R&D 税制）

#### 2.4.1 一般試験研究費の額に係る税額控除（措置法 42 の 4 第 1 項）

- 控除割合：試験研究費の 1% 〜 14%（増減試験研究費割合に応じてスライド）。
- 控除上限：法人税額の 25%（一般枠）+ 上乗せ枠（試験研究費割合 10% 超で最大 +10%、令和 8 年 3 月 31 日までは試験研究費割合 10% 超の上乗せ）。

#### 2.4.2 中小企業技術基盤強化税制（措置法 42 の 4 第 4 項）

- 対象：中小企業者等
- 控除割合：12% 〜 17%
- 控除上限：法人税額の 25%（試験研究費が前 3 年平均比 12% 超 or 売上に対する試験研究費 10% 超で +10% 上乗せ）

#### 2.4.3 オープンイノベーション型（特別試験研究費税額控除、措置法 42 の 4 第 7 項）

- 対象：大学・国立研究開発法人・特定中小企業者等との共同・委託研究
- 控除割合：20% 〜 30%（相手方により異なる）
- 控除上限：法人税額の 10%（一般枠と別枠）

**モデリング上の影響**

- **損益**：研究開発費（販管費）は full expense（日本基準）。
- **キャッシュフロー**：黒字化以降のみ効果。控除限度額管理が必要。
- **希薄化**：影響なし。
- **モデル組込**：「研究開発費」を独立行に分け、「税額控除可能額（=控除割合 × R&D 費用）」「税額控除上限（=法人税額 × 25% 等）」「実際適用額 = MIN(可能額, 上限)」で計算。

### 2.5 エンジェル税制（個人投資家向け、措置法 41 条の 19）

#### 2.5.1 4 つの優遇措置

| 措置 | 控除対象 | 控除上限 | 設立年数等 |
|---|---|---|---|
| プレシード・シード特例 | 株式譲渡益から控除 | 投資額 20 億円までは課税繰延なし、超過分は取得価額圧縮 | 設立 5 年未満 + 営業赤字 |
| 優遇措置 A | 総所得金額から控除 | (投資額 - 2,000 円) の所得控除（総所得金額 × 40% と 800 万円のいずれか低い額が上限） | 設立 5 年未満 |
| 優遇措置 B | 株式譲渡益から控除 | 投資額全額（譲渡益から控除） | 設立 10 年未満（一定要件） |
| 起業特例（プレシード・シード特例の発展形） | 自身が起業者の場合の譲渡益繰延 | 投資額 20 億円まで | 自己の創業時 |

#### 2.5.2 令和 7 年度改正（経産省 2025 年 7 月 1 日改訂ガイドライン）

- 払込期日が 2026 年 1 月 1 日以後の投資について、優遇措置 B およびプレシード・シード特例で「株式譲渡益が発生した翌年末まで（最大 2 年間）に投資した場合も適用可」へ拡張。

**モデリング上の影響（資金調達戦略）**

- **損益（投資先側）**：影響なし（投資家側の税制）。
- **発行側のメリット**：エンジェル投資家からの調達ハードル低下 → seed の調達金額・社数が増えやすい。
- **キャップテーブル**：エンジェル経由のシンプル株主が増えるため、後ラウンドでの SHA 整理コストに留意。

### 2.6 オープンイノベーション促進税制（法人投資家向け、措置法 66 の 13）

- 対象：国内事業会社または国内 CVC が、要件を満たすスタートアップの新規発行株式を一定額以上取得。
- 優遇内容：取得価額の 25% を所得控除（特別勘定として計上）。
- 対象スタートアップ要件（抜粋）：
  - 設立 10 年未満（一部 15 年）の未上場会社
  - 売上高研究開発費比率 10% 以上等の要件
- M&A 型（過半数取得）：従来 25% 控除。令和 8 年度改正でリニューアル予定。
  - 一括取得：25% 控除
  - 段階取得：20% 控除
  - 合併時の取崩し負担軽減

**モデリング上の影響**

- **発行側のメリット**：CVC からの調達金額の引き上げ要因に。
- **モデル組込**：CVC 調達のシナリオでは、CVC 側が控除を受けられる前提で valuation 交渉が成立しやすい。

### 2.7 ストック・オプション税制（措置法 29 条の 2）

#### 2.7.1 税制適格 SO の要件

| 要件 | 内容 | 改正 |
|---|---|---|
| 行使価額 | 付与時の株式時価以上（措置法 29 の 2 第 1 項 1 号） | 2023 年 7 月通達で未上場株の評価明確化（純資産価額方式・配当還元方式） |
| 行使期間 | 付与決議日の 2 年経過後 〜 10 年以内（一定スタートアップは 15 年）（同項 2 号） | 設立 5 年未満は 15 年特例（令和 5 年度改正） |
| 譲渡制限 | SO 自体の譲渡禁止（同項 3 号） | |
| 年間権利行使価額の上限 | 1,200 万円 → 最大 3,600 万円（令和 6 年度改正） | 設立 5 年未満：2,400 万円 / 設立 5 年以上 20 年未満の未上場・上場後 5 年未満：3,600 万円 |
| 株式管理 | 証券会社等への保管委託、または発行会社による株式の管理（令和 6 年度改正で発行会社による管理が認められた） | |
| 付与対象 | 自社・100% 子会社の取締役 / 使用人（社外高度人材は別枠） | |
| 1 人 1 個の付与制限 | 撤廃（実務上は問題なし） | |

> **2023 年 7 月通達の重要ポイント**：未上場株の SO 行使価額は、原則として「純資産価額方式」または「配当還元方式」（時価評価が困難な場合）を許容することが明確化された。これにより、直近ラウンド株価より低い行使価額の SO 設計が可能になり、税制適格 SO の利便性が大幅に向上した。

**経過措置**：2024 年 3 月 31 日以前に締結された契約については、2024 年 4 月 1 日 〜 12 月 31 日に契約変更すれば改正後要件を適用可能。

#### 2.7.2 税制適格 SO の課税

- 付与時：非課税
- 権利行使時：非課税（株式取得）
- 株式譲渡時：譲渡所得（分離課税 20.315% = 所得税 15% + 復興特別所得税 0.315% + 住民税 5%）

#### 2.7.3 税制非適格 SO（一般的な SO）

- 付与時：非課税
- 権利行使時：給与所得（時価 - 行使価額）として総合課税（最高税率 55.945%、所得税最高 45% + 復興特別 + 住民税 10%）
- 株式譲渡時：譲渡所得（行使時時価との差額）

#### 2.7.4 信託型ストックオプション（国税庁 2023 年 5 月見解）

**国税庁の見解**（令和 5 年 5 月 30 日「ストックオプションに対する課税（Q&A）」、同年 7 月最終改訂）：

- 信託型 SO の権利行使により役職員が株式を取得した場合の経済的利益は **給与所得**。
- 過去に行使済みの分も最大 5 年遡及して給与所得課税の対象。
- 信託組成段階で付与対象者を確定していないことが前提となる従来の譲渡所得扱いは否認。

**実務インパクト**

- 信託型 SO は事実上利用が困難に（既導入企業は税制適格 SO への切替・現金清算等の対応）。
- IPO 直前期に税務リスクが顕在化するケースが多発（東証の IPO 審査でも論点化）。

#### 2.7.5 有償ストックオプション

- 公正な評価額を払い込んで取得する SO。
- 権利行使時に課税なし。譲渡時に譲渡所得（20.315%）。
- 信託型 SO の問題が顕在化した後、代替手段として注目。
- ただしブラックショールズ評価の妥当性（評価が低すぎないか）が IPO 審査で論点化。

**モデリング上の影響（SO 全般）**

- **損益**：税制適格 / 非適格 / 信託型のいずれも会計上は付与時の公正評価額を費用化。税務上は永久差異が生じうる。
- **キャッシュフロー**：行使時は払込キャッシュ流入（行使価額 × 行使株数）。
- **希薄化**：付与済 SO は希薄化済み（fully diluted）扱い、未行使は将来希薄化要因。
- **インセンティブ・プールの想定**：日本の IPO 時点での SO プールは発行済株式の 10 〜 15% 程度が中央値。米国スタートアップ（15 〜 20%）よりやや低い。

### 2.8 暗号資産税制（参考：Web3 / トークンスタートアップ）

- 期末時点の保有暗号資産（自社発行 + 第三者発行）の **時価評価益への課税** が長年問題視されていたが、2023 年改正で「自己発行・継続保有・譲渡制限あり」のトークンは時価評価対象外に（法 61 条改正）。
- 令和 6 年度改正：第三者発行トークンも一定要件で時価評価対象外（短期売買目的でないもの）。
- 個人投資家のトークン売却益：雑所得（総合課税、最高 55%）。法人は分離課税の概念なし。

**モデリング上の影響**

- Web3 スタートアップの自社発行トークン保有は 2023 年改正以降、時価評価課税が回避可能に。
- IEO / TGE 戦略上、トークン発行価額・保有スケジュールが財務に与える影響を別途モデル化。

## 3. J-KISS（コンバーティブル・エクイティ）

### 3.1 J-KISS の本質と歴史

J-KISS は Coral Capital（旧 500 Startups Japan）が公開した、シードラウンド向けの **新株予約権型** 投資契約のテンプレート。2016 年初版（J-KISS 1.0）、2022 年 4 月に J-KISS 2.0 が公開された。米国の KISS / SAFE を日本の会社法・税制に合わせてアレンジしたもの。

| 区分 | 形態 | 特徴 |
|---|---|---|
| J-KISS（一般） | 新株予約権 | 適格資金調達 / M&A / 満期で株式に転換 |
| J-KISS-CB | 新株予約権付社債（少数派） | 利息・元本返還義務あり、シニア性 |

> 一般に「J-KISS」というときは新株予約権型を指す。新株予約権付社債型は別物。

### 3.2 標準条項（J-KISS 2.0）

#### 3.2.1 主要条項一覧

| 条項 | 標準値・標準仕様 | 備考 |
|---|---|---|
| 評価上限額（Valuation Cap） | post-money cap（J-KISS 2.0） | 1.0 では pre-money cap |
| 割引率（Discount） | 通常 20%（10 〜 30%） | 適格資金調達の発行価額からの割引 |
| 適格資金調達（Qualified Financing） | 1 億円以上のエクイティ調達（テンプレ） | 案件ごとに調整可 |
| 自動転換 | 適格資金調達発生時に自動転換 | |
| 任意転換 | 適格資金調達以外でも投資家任意で転換可（オプショナル） | |
| 満期（Maturity） | 18 カ月（テンプレ） | 満期に転換 or 償還 |
| 利息 | なし（テンプレ） | 法的には利息設計可 |
| 残余財産分配 | 出資額の優先分配（テンプレ） | M&A 時の処理 |
| MFN（Most Favored Nation） | あり | 後続 J-KISS の条件が有利な場合に同等条件適用 |
| 情報請求権 | 月次・四半期・年次の財務情報提供 | |

#### 3.2.2 J-KISS 1.0 vs 2.0：post-money cap への移行

J-KISS 1.0 の課題は、複数 J-KISS が積み上がるとき、起業家にとって希薄化が読みづらい点だった。J-KISS 2.0（2022 年 4 月公開）では valuation cap を pre-money から **post-money** に変更した。

| 観点 | J-KISS 1.0（pre-money cap） | J-KISS 2.0（post-money cap） |
|---|---|---|
| Cap の意味 | J-KISS 投資前の評価額の上限 | J-KISS 投資完了後の評価額の上限 |
| 起業家の希薄化見通し | 複数 J-KISS で読みづらい | 各ラウンドで明確に把握可能 |
| 投資家の権利保護 | 後続 J-KISS による希薄化が起こりうる | 自身の出資後の保有比率が確保される |

> 米国 SAFE も 2018 年 9 月に post-money cap に切替。J-KISS 2.0 はそれに追随。

### 3.3 適格資金調達時の転換計算

#### 3.3.1 転換価額の決定ロジック

J-KISS は次のいずれか **低い** 価額で転換される。

```
転換価額 = MIN(
    Discount Price,    # 適格資金調達発行価額 × (1 - Discount率)
    Cap Price          # Valuation Cap / 転換時発行済株式総数（post-money cap の場合）
)
```

#### 3.3.2 転換株式数

```
転換株式数 = J-KISS 払込金額 / 転換価額
```

**具体例**

| 前提 |
|---|
| J-KISS 投資額：1 億円 |
| 評価上限額（post-money cap）：5 億円 |
| 割引率：20% |
| 適格資金調達（Series A）：投資前評価額 8 億円、発行株価 80,000 円、発行済株式 10,000 株 |

```
Discount Price = 80,000 × (1 - 0.2) = 64,000 円
Cap Price = 500,000,000 / 10,000 = 50,000 円（pre-money cap 換算の場合）
              ※ post-money cap の場合は分母に転換後株式数を含むため反復計算

→ 転換価額 = MIN(64,000, 50,000) = 50,000 円
→ 転換株式数 = 100,000,000 / 50,000 = 2,000 株
```

> **post-money cap の正確な計算**：post-money cap は「J-KISS 転換後の評価額」を上限とするため、転換価額は「Cap / (転換後発行済株式総数)」となり、転換株式数を含む反復計算（または代数的解法）が必要。実務テンプレートでは Excel で goal seek を使うか、契約に計算式を明記する。

> **必ず両 version 併記 (E-H-009 解消)**: 上記の「簡略式 (Cap / 既存 FDSO)」は **pre-money 近似**であり、post-money cap で計算する場合と **5-20% の dilution 差** が出る。実務影響は (a) 複数 J-KISS が重なる場合、または (b) Cap が Series A pre-money に近い場合に大きい。したがって、post-money cap を採用しているのに pre-money 簡略式で算出すると founder 持分を 5-20pt 過大評価してしまう。本書の各数値例では「**iterative (post-money 正解) / 簡略 (pre-money 近似)**」を必ず両方提示する。`04b §12` の boundary 4-iteration trace と `15_input_schema §11` の `solve_post_money_cap()` を参照。

#### 3.3.3 複数 J-KISS の重なり処理

異なる cap / discount で複数の J-KISS が発行された場合、それぞれに転換価額を計算し、合計の転換株式数を Series A 発行株式数とは別に算出する。

| J-KISS | 投資額 | Cap | Discount | 転換価額 | 転換株数 |
|---|---|---|---|---|---|
| 第 1 回 | 50M 円 | 3 億 | 20% | 30,000 円 | 1,667 株 |
| 第 2 回 | 50M 円 | 5 億 | 20% | 50,000 円 | 1,000 株 |
| Series A 新規 | 200M 円 | - | - | 80,000 円 | 2,500 株 |
| 合計 | 300M 円 | | | | 5,167 株 |

**MFN 条項の影響**：第 2 回 J-KISS が第 1 回より投資家に有利な条件（より低い cap など）を含む場合、第 1 回投資家は MFN 条項により有利条件の適用を主張できる。

### 3.4 会計・税務上の取扱い

#### 3.4.1 会計処理（発行会社側）

- 払込金額を **新株予約権**（純資産の部）として計上。
- 利息計上は通常なし（J-KISS テンプレ）。

```
（払込時）
現金預金 100,000,000 / 新株予約権 100,000,000

（適格資金調達による転換時）
新株予約権 100,000,000 / 資本金 50,000,000
                          / 資本準備金 50,000,000
（資本金組入額は会社が選択。半額が資本準備金になることが多い）
```

#### 3.4.2 税務処理

- 発行時：非課税（払込金額が新株予約権の対価として認識）。
- 転換時：非課税（株式への転換は資本取引）。
- 失効・買戻時：失効益または買戻差額の処理が必要。

**モデリング上の影響**

- **損益**：通常、P/L 影響なし。
- **キャッシュフロー**：払込時に CF 流入。転換時に CF 影響なし。満期償還の場合は CF 流出。
- **希薄化**：J-KISS は転換まで cap table 上は新株予約権として表示。**希薄化計算時は post-money cap で換算した想定転換株数を含めるべき**。
- **勘定科目**：新株予約権（純資産の部、株主資本以外）。

### 3.5 J-KISS の満期処理

J-KISS テンプレでは、適格資金調達が発生せず満期（典型的に 18 カ月）を迎えた場合、次のオプションを提供する。

1. 満期日における株式（普通株式 or 約定の種類株式）への転換
2. 投資家の選択により償還（元本返還）

満期到来時の希薄化リスクと、ブリッジ調達としての利用は注意が必要（章 5.3 参照）。

### 3.6 J-KISS をブリッジ調達に使うときの留意点

- 直前ラウンドの株価を維持しながらブリッジ調達したい場合、Cap を直前ラウンドの post-money valuation に揃える運用が一般的。
- ただし下方ラウンド（ダウンラウンド）が起きた場合、J-KISS の Cap が高いため転換株式数が少なくなり、投資家にとってシニア性が確保されない問題が発生。
- Coral Capital は「ブリッジ用途では Cap をやや厳しめに設定する」「Discount を高めに設定する」運用を推奨。

## 4. 種類株式と株主間契約（SHA）

### 4.1 種類株式（会社法 108 条）

会社法 108 条に基づき、定款で定めることで発行できる種類株式は 9 種類。スタートアップで主に使うのは A 種・B 種等の優先株式。

| 種類 | 内容 | 会社法根拠 |
|---|---|---|
| 1. 剰余金配当 | 配当の優先・劣後 | 108 条 1 項 1 号 |
| 2. 残余財産分配 | 清算時の優先・劣後 | 同 2 号 |
| 3. 議決権制限 | 議決権事項の制限 | 同 3 号 |
| 4. 譲渡制限 | 譲渡承認条項 | 同 4 号 |
| 5. 取得請求権付 | 株主が会社に取得請求 | 同 5 号 |
| 6. 取得条項付 | 会社が一方的に取得 | 同 6 号 |
| 7. 全部取得条項付 | 株主総会決議で全部取得 | 同 7 号 |
| 8. 拒否権付（黄金株） | 特定事項の拒否権 | 同 8 号 |
| 9. 役員選任権付 | 取締役・監査役選任権 | 同 9 号（公開会社では不可） |

### 4.2 A 種優先株式の典型条項

| 条項 | 標準的内容 |
|---|---|
| 配当優先 | 1 株あたり投資単価 × 一定率（例：年 5%）の優先配当（実務では「みなし優先配当」で M&A 時に発動） |
| 残余財産分配優先 | 投資金額の 1 倍または 1.5 倍（×1, ×1.5 が日本の典型）+ 参加型 / 非参加型 |
| 議決権 | あり（普通株式と同等） |
| 取得請求権 | 上場時に普通株式へ転換、希薄化防止条項（フル・ラチェット or ブロードベース加重平均） |
| 取得条項 | IPO 時に普通株式へ自動転換 |
| 全部取得条項 | M&A スクイーズアウト等で活用 |
| みなし清算条項 | M&A を清算とみなして残余財産分配優先を発動（SHA に記載が一般的） |

#### 4.2.1 残余財産分配優先（Liquidation Preference）

```
ケース A：1× 非参加型 + みなし清算
  M&A 売却額 200M 円、A 種優先 100M 円投資（25% 持分相当の発行）
  → A 種：MAX(投資金額 100M, 売却額 × 持分比率 50M) = 100M 円
  → 普通株主：100M 円（残余）

ケース B：1× 参加型 + みなし清算
  同条件
  → A 種：投資金額 100M（優先分配） + 残額 100M × 25%（持分比率） = 125M 円
  → 普通株主：100M × 75% = 75M 円

ケース C：1× 参加型・キャップ付（cap 3×）
  売却額が大きい場合は転換した方が有利になる閾値（Conversion Point）を計算
```

> **日本での実務トレンド**：1.0 〜 1.5 倍 + 非参加型が主流。ただし、ダウンラウンド・ブリッジでは 2 倍参加型 + キャップ付きを使うこともある（投資家保護のため）。

#### 4.2.2 希薄化防止条項（Anti-dilution）

| 方式 | 計算式 | 投資家保護度 |
|---|---|---|
| フル・ラチェット | 転換価額 = 後続ラウンド発行価額 | 強（最強） |
| ブロードベース加重平均 | 転換価額調整 = 旧転換価額 × (発行済 + 新規発行 / 発行済 + 新規発行 × 旧価額 / 新価額) | 中 |
| ナローベース加重平均 | 同上、分母を転換可能株式に限定 | 中強 |

日本の実務ではブロードベース加重平均が圧倒的多数。

### 4.3 株主間契約（SHA）の標準条項

#### 4.3.1 主要条項一覧

| 条項 | 標準的内容 | スタートアップへの影響 |
|---|---|---|
| 取締役指名権 | A 種投資家がリード VC として 1 〜 2 名指名 | ガバナンスコスト |
| オブザーバー権 | 取締役会への参加・発言権（議決権なし） | 情報共有負担 |
| 重要事項拒否権 | 重要事項（資金調達 / M&A / 役員報酬等）に投資家承諾必要 | 経営の自由度低下 |
| 先買権（ROFR） | 既存株主が他株主の譲渡時に優先買取 | キャップテーブル管理コスト |
| 共同売却権（タグ・アロング） | 創業者が売却するとき投資家も同条件で売却できる | 起業家のセカンダリ制約 |
| 強制売却権（ドラッグ・アロング） | 多数派が売却決定すると少数派も同条件で強制売却 | M&A スムーズ化 |
| MFN（最恵国待遇） | 後続投資家に有利条件があれば自動適用 | 投資家保護 |
| 情報請求権 | 月次・四半期・年次の財務情報提供 | 報告コスト |
| 競業避止 | 創業者の競業禁止 | 起業家の制約 |
| みなし清算 | M&A を清算とみなす | 種類株式の優先分配と連動 |
| ロックアップ（IPO） | IPO 時に一定期間譲渡制限 | 投資家のエグジット制約 |

#### 4.3.2 タグ・アロング vs ドラッグ・アロング

- **タグ・アロング（Tag-Along Right）**：少数株主に「同条件で売る権利」を与える。少数派保護。
- **ドラッグ・アロング（Drag-Along Right）**：多数派が売却を決めたとき少数派に「同条件で売却を強制する」権利。M&A の交渉円滑化。

**注意点**：起業家がマイノリティに転落した状況でドラッグ・アロングが発動されると、起業家が望まない条件で会社を売却する事態になる。日本のスタートアップ実務では発動条件（持分比率・最低価格）を厳格に設計するのが通例。

### 4.4 VC スタンダード（独立系 vs CVC）

| 区分 | 典型的な条件 | 備考 |
|---|---|---|
| 独立系 VC（ANRI / Coral / Globis 等） | 1.0 〜 1.5x 非参加型、ブロードベース加重平均、取締役 1 名 + オブザーバー | グローバル標準に近い |
| 大手金融系 VC（JAFCO / SBI / DBJ 等） | 1.5 〜 2x、参加型もあり、創業者保証要求の場合あり | 比較的伝統的 |
| CVC（KDDI∞ / Sony IF / NTT Docomo Ventures 等） | 親会社事業との戦略的関係条項（優先交渉権・ライセンス等）、純粋投資条件は緩め | 戦略的シナジー重視 |
| 海外 VC（Sequoia / DCM / Coatue 等） | グローバル標準、非参加型、加重平均 | NDA / 開示要件が厳格 |

**モデリング上の影響**

- **損益**：直接影響なし。
- **キャッシュフロー**：直接影響なし。
- **希薄化**：種類株条項により M&A 時のリターン分配が大きく変動。発行済株式比率では普通株主 / 種類株主の見かけ持分とエグジット時の実質リターンが乖離するため、必ずエグジットシナリオごとの分配シミュレーションを構築。
- **モデル組込**：waterfall 分析（M&A バリュエーション × 種類株式分配 → 各株主のリターン）を別シートで構築。

### 5.2 日本政策金融公庫（JFC）創業融資

#### 5.2.1 新規開業・スタートアップ支援資金（旧：新規開業資金）

2024 年 3 月に「新創業融資制度」は廃止され、「新規開業・スタートアップ支援資金」に統合（2025 年 3 月に名称変更）。多くの新規開業者が無担保・無保証で利用可能。

| 項目 | 内容 |
|---|---|
| 融資限度額 | 7,200 万円（うち運転資金 4,800 万円） |
| 返済期間 | 設備資金 20 年以内（うち据置 5 年以内）、運転資金 10 年以内（うち据置 5 年以内） |
| 担保・保証 | 原則として無担保・無保証（一部条件あり） |
| 基準利率（2026 年初時点、無担保） | 年 3.25 〜 4.65%（変動） |
| 特別利率 A（無担保） | 年 2.45 〜 4.05% |
| 基準利率（有担保） | 年 1.55 〜 3.65% |

**特別利率の対象**：女性、35 歳未満、55 歳以上、技術・ノウハウ等に新規性のある事業、産業競争力強化法の認定創業者等。

#### 5.2.2 資本性ローン（挑戦支援資本強化特別貸付）

- 自己資本としてみなされる劣後ローン（金融機関の融資審査上）。
- 期間 5 年 1 カ月〜 20 年。元本は満期一括返済（期中は利息のみ）。
- 業績連動金利（赤字時は最低利率、黒字時は最高利率）。
- スタートアップ：限度額 1.5 億円。

**モデリング上の影響**

- **損益**：支払利息（営業外費用）。
- **キャッシュフロー**：当初に元本受領（財務 CF +）、毎期利息支払（営業 CF -）、満期に元本返済（財務 CF -）。
- **希薄化**：影響なし（負債）。
- **勘定科目**：長期借入金 / 1 年以内返済予定長期借入金 / 支払利息。

### 5.3 典型的キャップテーブル進化（モデル例）

#### モデル前提
- 創業者：100% 普通株式 10,000 株
- シード：J-KISS 100M 円、Cap 5 億円、20% Discount
- シリーズ A：A 種 200M 円、Pre-money 8 億円
- シリーズ B：B 種 1,000M 円、Pre-money 30 億円
- IPO：50 億円調達、Pre-money 200 億円

| 段階 | 創業者持分 | シード投資家 | A 種 | B 種 | IPO 時投資家 | SO プール |
|---|---|---|---|---|---|---|
| 創業時 | 100% | - | - | - | - | - |
| シード（J-KISS 発行、未転換） | 100% | （新株予約権） | - | - | - | - |
| シリーズ A（J-KISS 転換 + A 種発行） | 約 64% | 約 12% | 約 19% | - | - | 約 5% |
| シリーズ B 後 | 約 50% | 約 9% | 約 15% | 約 20% | - | 約 6% |
| IPO 時（自動転換 + 新規発行） | 約 30% | 約 5% | 約 9% | 約 12% | 約 35% | 約 9% |

> **創業者の IPO 時持分の中央値**：日本のグロース上場では創業者（共同創業含む）持分は 15 〜 30% が中央値。米国（5 〜 15% 程度）より高い。理由：日本のグロース上場は早期段階で IPO 可能（時価総額 5 億円以上）、結果として調達総額・希薄化が小さい。

### 5.4 IPO 時の自動転換

種類株式（A 種、B 種等）はすべて、IPO 上場承認決議の時点で **普通株式に自動転換**（取得条項発動）。転換比率は通常 1:1（=投資単価ベース）だが、希薄化防止条項が発動した場合は調整が入る。

**モデリング Tips**

- 上場直前期の cap table は「種類株式表記」「普通株式換算（自動転換後）」の 2 通りで管理。
- 種類株主間の優先分配権は IPO 時消滅 → IPO エグジットでは普通株主と同等のリターン。M&A エグジットのみ優先分配が発動。

### 5.5 ロックアップ

- 主幹事証券との合意により、IPO 時の創業者・主要株主・VC 等は通常 90 日 〜 180 日のロックアップを設定。
- グロース市場：「上場日から 6 カ月（180 日）」が標準。
- ロックアップ解除条件：時間経過、または株価が公募価格の 1.5 倍以上で 5 営業日連続等。

## 7. スタートアップ特有論点

### 7.1 役員報酬の税務

#### 7.1.1 損金算入が認められる 3 種類（法 34 条）

| 種類 | 内容 | 損金算入要件 |
|---|---|---|
| 定期同額給与 | 毎月おおむね一定額の給与 | 事業年度開始から 3 カ月以内の改定（臨時改定事由・業績悪化改定事由を除く） |
| 事前確定届出給与 | 所定時期に確定額（金銭・株式・SO）支給 | 株主総会決議から 1 カ月以内 or 事業年度開始から 4 カ月以内のいずれか早い日までに税務署へ届出 |
| 業績連動給与 | 業績指標に連動する給与 | 同族会社（非上場含む）は適用不可。上場会社向け |

中小企業・非上場会社では、定期同額給与 + 事前確定届出給与の組合せが基本。

#### 7.1.2 役員報酬設定の実務上の留意点

- **改定時期の制約**：定期同額給与は事業年度開始 3 カ月以内に改定。それ以外の時期に改定すると一部損金不算入。
- **社会保険料負担**：役員報酬月額に応じて健康保険・厚生年金保険料が会社・本人で発生。スタートアップでは月額 80 〜 100 万円前後で社会保険料コストが顕著に増える。
- **役員退職慰労金**：法 34 条 1 項 2 号により、不相当に高額な部分は損金不算入。功績倍率法（最終月額報酬 × 在任年数 × 功績倍率）が定石、功績倍率は社長 3.0、専務 2.4、常務 2.2 程度が目安。

**モデリング上の影響**

- **損益**：役員報酬は販管費（人件費）。改定タイミングを誤ると損金不算入で実効税率が見かけ上上昇。
- **キャッシュフロー**：報酬支払 + 社会保険料負担。
- **希薄化**：影響なし（株式報酬の場合は別途）。

### 7.2 資本金の額：1 億円・5 億円の壁

| 資本金額 | 主な変化 | スタートアップへの影響 |
|---|---|---|
| 1,000 万円以下 | 設立時消費税免税 2 年（インボイス登録すれば免税撤回） | 設立後 2 年の消費税優遇 |
| 1,000 万円超 | 設立 2 年目から消費税課税事業者 | キャッシュフロー悪化 |
| 1 億円以下 | 中小法人税制適用（軽減税率・100% 欠損金控除・少額減価償却資産 etc） | 税制優遇継続のため、IPO 直前まで 1 億円以下を維持するのが定石 |
| 1 億円超 | 外形標準課税の対象、欠損金控除限度 50%、軽減税率不適用 | 増資戦略で「資本金 + 資本準備金」の組入バランスを設計 |
| 5 億円以上 | 会社法上の大会社、会計監査人設置義務、損益計算書公告義務 | 監査コスト・統制コスト増大 |

#### 7.2.1 資本金組入額の調整（会社法 445 条）

会社法上、株式発行時に払込金額の 1/2 以上を資本金に組み入れる義務があるが、最大で 1/2 を資本準備金に振り分け可能。

```
例：100M 円の調達
- 資本金 50M、資本準備金 50M：資本金は 50M 増加のみ
- 資本金 100M、資本準備金 0：資本金が 100M 増加
```

**戦略**：資本金が 1 億円を超えそうな調達を控える場合、資本準備金への組入を最大化することで税制優遇を維持。

### 7.3 少額減価償却資産特例（措置法 67 条の 5）

- 中小企業者等が 30 万円未満の減価償却資産を取得し事業供用した場合、年間 300 万円を限度として全額損金算入可能。
- 適用期限：令和 8 年 3 月 31 日（2026 年 3 月 31 日）まで。
- 令和 8 年度税制改正大綱で **40 万円未満に引上げ + 令和 11 年 3 月 31 日まで 3 年延長** が示された（法案成立後反映）。
- 対象法人要件：資本金 1 億円以下 + 常時使用従業員数 500 人以下。

**モデリング上の影響**

- **損益**：減価償却を前倒し、初期赤字スタートアップでは繰越欠損金を増やすことで、将来黒字化期の節税効果を最大化。
- **キャッシュフロー**：黒字化以降に法人税減で CF +。
- **希薄化**：影響なし。
- **モデル組込**：CapEx を「30 万円未満（即時損金）」「30 万円以上（資産計上 + 通常減価償却）」に分けて入力。

### 7.4 消費税

#### 7.4.1 課税事業者・免税事業者の判定

- 基準期間（前々事業年度）の課税売上高が 1,000 万円超 → 課税事業者
- 設立 1 期目・2 期目：原則として免税事業者（資本金 1,000 万円未満かつ特定要件を満たす場合）

#### 7.4.2 簡易課税制度（消費税法 37 条）

- 基準期間の課税売上 5,000 万円以下の事業者が選択可能。
- 業種別「みなし仕入率」（卸売 90%、小売 80%、製造 70%、その他 60%、サービス 50%、不動産 40%）を売上消費税に乗じて納付額を算出。
- 適用は事前届出 + 2 年継続義務。

#### 7.4.3 インボイス制度（適格請求書等保存方式、2023 年 10 月 1 日 〜）

**核心**：仕入税額控除を行うためには、適格請求書発行事業者の発行する適格請求書（インボイス）の保存が必須。

#### 7.4.4 経過措置のロードマップ

| 期間 | 内容 |
|---|---|
| 2023 年 10 月 〜 2026 年 9 月 | 2 割特例（売上税額の 2 割を納付）、免税事業者からの仕入は 80% 控除 |
| 2026 年 10 月 〜 2029 年 9 月（令和 8 年度改正） | 個人事業主のみ 3 割特例（法人は終了）、免税事業者からの仕入は 70% 控除へ縮小 |
| 2029 年 10 月以降 | 経過措置終了、免税事業者からの仕入は控除不可 |

#### 7.4.5 スタートアップへの影響

- **B2B SaaS / 受託**：取引先（課税事業者）が控除できない不利益を回避するため、インボイス登録が事実上必須に。
- **C2C / コミュニティ系**：個人クリエイターが免税事業者として残る場合、プラットフォーマーは仕入控除損失を吸収するか、価格設計を変える必要。
- **消費税還付**：輸出取引・大型 CapEx 投資時は還付対象。原則課税が有利になるケースあり。

**モデリング上の影響**

- **損益**：消費税は税抜経理が一般的。免税期は売上消費税が利益に取り込まれるため、課税転換時に「見かけの売上減」が発生（税抜化に伴う）。
- **キャッシュフロー**：中間納付（年税額が一定額超）+ 確定納付。
- **モデル組込**：「課税事業者転換タイミング」「2 割特例 / 簡易課税 / 原則課税の選択」を可変パラメータ化。

### 7.5 電子帳簿保存法（電帳法、2024 年 1 月本格化）

#### 7.5.1 改正の核心

- 電子取引（メール添付の請求書、EC サイトの領収書等）は **電子データのまま保存が義務化**（2024 年 1 月 1 日 以降、宥恕期間終了）。
- 2024 年 1 月以降は「相当の理由」がある場合の猶予措置あり（恒久化）。

#### 7.5.2 主要要件

- **真実性の確保**：タイムスタンプ付与（2 カ月 + 7 営業日以内）、訂正削除履歴の保存、または事務処理規程の備置。
- **可視性の確保**：取引年月日 / 取引金額 / 取引先 の 3 項目で検索可能に。

#### 7.5.3 スタートアップの実務対応

- クラウド会計（freee / マネーフォワード / 弥生）+ 経費精算 SaaS（楽楽精算 / Bill One / Concur）の組合せが標準。
- IT 導入補助金で初期コスト圧縮可能（最大 450 万円）。

**モデリング上の影響**

- **損益**：会計 SaaS 費用（販管費）。月額 5 〜 30 万円規模。
- **キャッシュフロー**：年契約前払いが多い。
- **勘定科目**：通信費 / ソフトウェア利用料 / 支払手数料。

### 7.6 ESOP 信託 vs ストックオプション

| 項目 | ストックオプション | ESOP（信託型インセンティブ） |
|---|---|---|
| 対象 | 役員・従業員（要件次第で取引先・社外高度人材も） | 役員・従業員 |
| 仕組み | 行使価額で株式取得する権利 | 信託が株式を保有、業績達成等で受益権付与 |
| 課税 | 適格 SO：譲渡時 20.315% / 非適格：行使時総合 / 信託型：給与所得（2023 年見解） | 信託受託会社の処理次第（給与・賞与扱いの場合多い） |
| 希薄化 | 新株発行で希薄化 | 既存株式を信託保有 → 既存株主の希薄化なし（自社株買いが原資） |
| キャッシュ消費 | 行使時に払込キャッシュ流入 | 信託への株式拠出時に CF 流出（自己株式取得） |
| 上場準備期 | 主流 | 上場会社が多い |

**モデリング上の影響**

- ESOP は「自己株式（負の純資産）」+「信託拠出 CF」がキャッシュフロー上発生。
- スタートアップでは IPO 後の従業員報酬として導入することが多く、IPO 前の主役は依然として税制適格 SO。

## 8. IPO 準備

### 8.1 全体スケジュール

```
N-3 期: ショートレビュー、上場準備プロジェクト発足
        ↓
N-2 期: 監査法人による準金商法監査開始、主幹事証券選定、内部統制整備
        ↓
N-1 期: 期首から上場会社並みの統制運用、業績数値の堅実な達成
        ↓
N 期: 主幹事証券審査 → 取引所審査 → 上場
```

### 8.2 各期の主な対応事項

#### 8.2.1 N-3 期（直前々々期）

- **ショートレビュー**：監査法人による事前調査。論点の洗い出し（収益認識、SO、関係会社、組織再編、コーポレートガバナンス、会計方針）。
- **資本政策の固定化**：上場準備に入った後の追加調達は希薄化制限を受けるため、シリーズ B 〜 C で大型調達を完了させる。
- **関係会社整理**：オーナー個人と会社の取引（不動産賃貸、貸付金、保証等）の解消。
- **役員退職慰労金規程**の整備、過年度の役員退職慰労金の妥当性確認。

#### 8.2.2 N-2 期（直前々期）

- **準金商法監査**：監査法人による正式監査開始。期首日付で会計方針の見直し、過年度修正再表示。
- **主幹事証券選定**：野村 / 大和 / SMBC 日興 / みずほ / 三菱 UFJ モルガン・スタンレー が大手 5 社。スタートアップは SBI / 楽天 / マネックス 等もよく利用。
- **内部統制システムの構築**：稟議規程、職務分掌、業務フロー、リスクコントロールマトリクス（RCM）整備。
- **取締役会・監査役会の運用**：月次取締役会、議事録整備、監査役監査計画。
- **関連当事者取引の整理**：100% 子会社化、撤退、市場価格による取引化。

#### 8.2.3 N-1 期（直前期）

- **期首から上場会社並みの運用**：取締役会・監査役会・指名委員会・報酬委員会の実効的運営。
- **主幹事証券審査**：四半期業績達成、月次経営会議、KPI モニタリング。
- **コーポレート・ガバナンス報告書** ドラフト作成。
- **業績予想の精度向上**：四半期予実差異 ±10% 以内に抑える運用力。
- **N-1 期決算 → 短信 + 有価証券届出書（I の部・II の部）作成**。

#### 8.2.4 N 期（申請・上場期）

- **上場申請** → **取引所審査**（書面審査 + ヒアリング 3 〜 4 回）→ **承認** → **上場**。
- **公募・売出**：ブックビルディング方式が主流。仮条件 → 公募価格決定（投資家ヒアリング）。
- **ロードショー**：投資家プレゼン（国内・海外）。
- **上場日**：初値形成 → 監督官庁へ届出。

### 8.3 監査法人選定

- **大手 4 法人**：EY 新日本、あずさ、トーマツ、PwC Japan
- **準大手**：太陽、仰星、PwC 京都、東陽 等
- **中堅・スタートアップ特化**：A&A パートナーズ、UHY 東京、HLB Meisei 等

スタートアップは大手 4 法人に偏重しがちだが、近年は準大手・中堅の方が機動的でコストが低い。ただし IPO 規模・グローバル性によっては大手が必要。

### 9.2 補助金の会計処理

#### 9.2.1 圧縮記帳（法 42 条、43 条等）

固定資産取得を補助金で行う場合、圧縮記帳により取得時の課税を繰延可能。

**直接減額方式**（中小法人実務で多い）：

```
（補助金受領時）
現金預金 50,000,000 / 雑収入（益金算入）50,000,000

（固定資産取得時）
機械装置 50,000,000 / 現金預金 50,000,000
固定資産圧縮損 50,000,000 / 機械装置 50,000,000
（圧縮損 = 損金算入）
→ 帳簿価額 0 円、以後減価償却なし
```

**積立金方式**（上場会社で多い）：

```
（受領 + 取得後）
（補助金収益と取得費は通常通り計上、減価償却も通常通り）
（決算時に剰余金処分で）
繰越利益剰余金 / 圧縮積立金
（以後、減価償却に応じて取り崩し → 課税所得への加算）
```

#### 9.2.2 補助金収入の会計上の扱い（収益認識）

- 通常は「雑収入」または「営業外収益」または「特別利益」（金額重要性に応じて）。
- 収益認識基準上、政府補助金は IFRS / 日本基準とも企業会計基準 第 29 号の対象外（補助金会計基準の議論は別途進行中）。

**モデリング上の影響**

- **損益**：補助金は通常、営業外収益または特別利益。EBITDA / 営業利益から除外される。
- **キャッシュフロー**：交付決定 → 実績報告 → 入金まで時間差大（3 〜 12 カ月）。CF タイミングを別途モデル化。
- **税効果**：圧縮記帳で課税繰延、積立金方式の場合は将来の取崩しで課税。

## 10. M&A 関連の日本特有論点

### 10.1 株式交付制度（2021 年 3 月施行、会社法 774 条の 2 〜 11）

#### 10.1.1 制度概要

- 2021 年 3 月施行の改正会社法で新設された M&A 手法。
- 株式 **会社（買収者）が、対象会社の株式を譲り受け、対価として自社の株式を交付**する手続き。
- 対象会社の株式の **過半数取得が要件**（子会社化が前提）。
- 株式交換（100% 子会社化）と株式譲渡（個別契約）の中間的手法。

#### 10.1.2 税制（措置法 37 条の 13 の 3、66 条の 2 の 2）

- 対象会社株主：株式交付対価のうち、**親会社株式の価値が対価総額の 80% 以上** であれば、譲渡損益の繰延（適格扱い）。
- 買収者：取得した子会社株式の取得価額 = 交付した自社株式の時価。

#### 10.1.3 株式交換 / 株式移転 / 株式交付の比較

| 項目 | 株式交換 | 株式移転 | 株式交付 |
|---|---|---|---|
| 制度導入 | 1999 年 | 1999 年 | 2021 年 |
| 結果 | 100% 子会社化 | 共同持株会社化 | 過半数子会社化 |
| 対価 | 株式・金銭等 | 株式 | 株式（一部金銭可） |
| 対象会社の株主総会決議 | 必要（特別決議） | 必要（特別決議） | **不要**（買収者側のみ特別決議） |
| 反対株主の買取請求 | あり | あり | あり |
| 適格要件 | 法人税法 62 条の 9 等 | 同上 | 措置法 37 の 13 の 3 |
| スタートアップ M&A での主用途 | 100% 取得・上場会社による完全子会社化 | 持株会社化 | 株式対価 M&A、子会社化（過半数） |

#### 10.1.4 スタートアップでの活用

- **エグジット**：上場会社による株式対価買収 → スタートアップ株主が上場会社株式を受領 → 譲渡益繰延（保有継続中）。
- **買収側として**：CapEx 不足のスタートアップが、自社株式を対価に他社買収。

**モデリング上の影響**

- **損益**：のれんの発生（買収プレミアム）、20 年以内償却。
- **キャッシュフロー**：株式対価のため CF 流出は最小。アドバイザリー・税務・登記費用のみ。
- **希薄化**：自社株式の新規発行 → 希薄化発生。
- **勘定科目**：のれん（無形固定資産）、子会社株式（投資有価証券）、資本金 / 資本準備金。

### 10.2 適格組織再編税制（法 62 条の 2 〜 8 等）

#### 10.2.1 適格判定の主要要件

| 要件区分 | 主要内容 |
|---|---|
| 完全支配関係（100%） | 金銭等不交付 + 完全支配関係継続 |
| 支配関係（50% 超） | 金銭等不交付 + 支配関係継続 + 主要事業継続 + 従業者引継 |
| 共同事業（50% 以下、グループ外） | 上記 + 事業関連性 + 規模要件 + 経営参画 + 株式継続保有 |

適格判定の結果：

- **適格**：簿価引継、譲渡損益繰延、繰越欠損金引継（一部制限）。
- **非適格**：時価評価、譲渡益課税。

#### 10.2.2 繰越欠損金の引継制限

- 適格組織再編でも、被合併会社の繰越欠損金引継には「**みなし共同事業要件**」の充足または **支配関係 5 年要件** の充足が必要。
- ロールアップ型 M&A（中小企業統合）では、繰越欠損金が引き継げないケースが多い。

### 10.3 TOB（株式公開買付、金商法 27 条の 2）

#### 10.3.1 強制 TOB のトリガー

- 上場会社の株式を市場外で 5% 超取得 → TOB 強制（短期間内に 10 名以上から取得など）。
- 1/3 ルール：1/3 を超える買付は TOB 必須。
- 2/3 ルール：2/3 を超える買付は全部買付義務。

#### 10.3.2 スタートアップでの関連性

- 上場後の創業者・VC の売却（5% 超ブロック取引）。
- 上場会社による上場スタートアップの買収（クロスボーダー含む）。

### 10.4 スクイーズアウト

#### 10.4.1 主要手法

| 手法 | 仕組み | 主な活用場面 |
|---|---|---|
| 株式売渡請求（会社法 179 条） | 90% 以上保有株主が他株主に売渡請求 | 完全子会社化（90% 取得後） |
| 全部取得条項付種類株式 | 全部取得条項を付した上で取得 → 端数処理 | TOB 後の少数株主排除 |
| 株式併合 | 株式併合で少数株主を端数化 → 端数処理 | 同上 |
| 株式交換 | 100% 株式交換 | 上場会社による完全子会社化 |

#### 10.4.2 スタートアップ M&A での活用

- VC 主導の M&A エグジット時に、創業者の少数派が反対 → 株式併合 + 端数処理で強制排除（極めて稀）。
- ドラッグ・アロング条項（章 4.3）の発動と並行することが多い。

### 10.5 会社分割

- 吸収分割：既存会社が事業を承継。
- 新設分割：新設会社が事業を承継。
- 適格判定は組織再編税制と同じ枠組み。
- スタートアップでの活用：主力事業のスピンアウト、ノンコア事業の切り離し。

### 11.2.1 CVC 受け入れ判定ルール (E-H-038 解消)

**CVC を Lead 投資家として受け入れる条件 (3 つすべて充足が望ましい):**

1. **戦略的関係が round の primary value**: 親会社との販路・データ・流通アクセスが、フィナンシャル投資家からは得られないレベルで重要 (例: 通信キャリアと連携必須の IoT スタートアップ → KDDI ∞ Labo)。単なる「将来の M&A 候補」では不十分。
2. **契約条項に sunset / exclusivity carve-out 明記**: 優先交渉権 / 独占ライセンスは期間 (3-5 年) と地域 (日本国内のみ等) を限定。Founder consent 付き drag-along と組み合わせ、CVC の親会社事業の方針転換で startup が縛られないように防御。
3. **独立系 VC が最低 1 社 same round に参加**: 純財務 return を defend する独立系の board observer を確保。CVC 単独 lead は valuation や exit 判断が親会社事業都合に偏るリスク。

**CVC 受け入れを避ける signal**: 親会社事業との将来的競合可能性、ROFR (Right of First Refusal) on M&A の独占要求、optional 親会社株式 swap の要求、or 親会社経営方針の急変リスク (= 親会社の業績悪化 / トップ交代頻度高) — これらは VC 純財務 vs CVC 戦略の trade-off の片側に過度に倒れる兆候。

- VC 出資比率は累積で 60 〜 70% が IPO 直前の典型値。

## 12. 為替・国際展開

### 12.1 海外子会社設立の選択肢

| 構造 | 概要 | 主な利用シーン |
|---|---|---|
| **日本本社 + 海外子会社** | 日本本社配下に Delaware Inc / Singapore Pte Ltd 等 | 海外営業拠点、米国ブランチ |
| **Delaware Flip** | 米国 Delaware に新設持株会社、日本会社を子会社化 | YC アクセラ、米国 VC からの調達、米国上場視野 |
| **Singapore Holdco** | シンガポール持株会社、日本会社を子会社化 | アジア展開、低税率、特許・IP 管理 |
| **Cayman / BVI Holdco** | ケイマン諸島・英領 V.I. に持株会社 | クロスボーダー M&A、暗号資産・ファンド構造 |

### 12.2 Delaware Flip の論点

#### 12.2.1 Flip プロセス

1. Delaware Inc 新設
2. 日本会社株主が日本会社株式を Delaware に出資 → Delaware Inc が日本会社の親会社化
3. 既存契約（VC 投資契約、SHA、SO、雇用契約）の対応関係を整理

#### 12.2.2 主要論点

- **税務**：日本株主の Delaware 株式出資は譲渡所得課税のトリガー（時価評価）。タックスフリー化は限定的（適格現物出資要件）。
- **知財・契約**：日本会社の知財を Delaware に移転する場合、移転価格税制の対象（次節）。
- **CFC（タックスヘイブン対策税制）**：Delaware Inc が CFC 該当 → 日本側で合算課税の可能性。
- **PE 認定**：日本側に管理機能が残ると Delaware Inc が日本に PE を有する判定。

### 12.3 Tax Treaty と PE 認定リスク

#### 12.3.1 PE（恒久的施設）の類型（OECD モデル / 各国租税条約）

- 物理的 PE：支店、事務所、工場
- 工事 PE：12 カ月超の建設プロジェクト
- 代理人 PE：従属代理人が反復的に契約締結

#### 12.3.2 スタートアップでの典型的論点

- **海外駐在員**：日本本社から駐在員を派遣 → 駐在員が現地で契約締結権を持つと **代理人 PE 認定リスク**。
- **リモートワーク従業員**：海外居住従業員が顧客対応を行う → 同様の PE リスク。
- **クラウドサーバー**：原則として PE には該当しないが、専用サーバー + 業務遂行で論点化（コミットメント次第）。

**実務対応**：契約締結権を本社に集約、駐在員業務を補助・サポートに限定、現地法人化の検討。

### 12.4 移転価格税制（措置法 66 条の 4）

#### 12.4.1 概要

- 国外関連者との取引は **独立企業間価格（Arm's Length Price）** で行う必要。
- 違反時：所得移転とみなして日本側で課税、二重課税は租税条約上の相互協議で解消。

#### 12.4.2 スタートアップでの論点

- 日本本社から海外子会社への **マネジメントサービス・知財ライセンス**：適切なロイヤリティ料率を設定。
- **コストプラス法**：費用 + 適切なマークアップ（例：5 〜 10%）。
- **取引単位営業利益法（TNMM）**：機能・リスクに応じた利益水準。

**文書化義務**：直前年度の連結売上 1,000 億円超は CbCR（国別報告書）等の義務あり。スタートアップ規模では基本的に対象外だが、ローカルファイル相当の文書化は推奨。

### 12.5 CFC（タックスヘイブン対策税制 / 外国子会社合算税制、措置法 66 条の 6）

#### 12.5.1 概要

- 日本居住者・内国法人が直接間接 50% 超を保有する外国法人（外国関係会社）が対象。
- 外国関係会社の所得を日本側で合算課税する場合あり。

#### 12.5.2 判定フレーム（2017 年改正以降）

```
Step 1: 外国関係会社の判定（保有比率 50% 超 or 実質支配）
Step 2: 経済活動基準の判定
   - 事業基準（持株業 / 知財業等は基本不適合）
   - 実体基準（事務所等の物理的存在）
   - 管理支配基準（事業管理が現地）
   - 関連者取引基準 / 非関連者基準（地域差あり）
Step 3: 各基準を全て満たす → 対象会社株式から除外（部分合算 / 受動所得のみ合算へ）
Step 4: 各基準を満たさない → 全所得合算課税
```

#### 12.5.3 ペーパーカンパニー判定

ペーパーカンパニー / キャッシュボックス / ブラックリスト国所在の外国関係会社は、税率にかかわらず合算対象（外国会社合算課税の対象）。

**スタートアップでの実務**

- Singapore Holdco / Delaware Inc を持株目的のみで設立すると、ペーパーカンパニー判定リスク。
- 実質的な事業活動・人員配置・現地管理が必要。

### 12.6 為替リスクのモデル化

#### 12.6.1 換算方法（外貨建取引等会計処理基準）

- 取引時点：取引日レート
- 期末時点：金銭債権債務は決算時の為替レートで換算（為替差損益発生）
- 在外子会社：B/S は決算時レート、P/L は期中平均レート、差額は為替換算調整勘定（純資産）

#### 12.6.2 ヘッジ会計

- 為替予約：ヘッジ会計適用で為替変動リスクを相殺。
- スタートアップでは月次キャッシュフロー予測 + 為替予約の組合せが基本。

**モデリング上の影響**

- **損益**：為替差損益（営業外）。
- **キャッシュフロー**：海外子会社からの配当受取（外国子会社配当益金不算入：受取配当の 95% が益金不算入）。
- **希薄化**：影響なし。
- **モデル組込**：為替シナリオ（円安・円高）別の感応度分析。

    nol_used = min(prior_nol_balance, nol_limit)
    taxable_after_nol = taxable_income - nol_used
    corp_tax_after_nol = ...  # 再計算

    # 3. 試験研究費税額控除（簡易：上限 25%）
    rd_credit = min(rd_expense * 0.10, corp_tax_after_nol * 0.25)
    corp_tax_after_credit = corp_tax_after_nol - rd_credit

    local_corp_tax = corp_tax_after_credit * 0.103

    # 5. 防衛特別法人税（2026/4 以降）
    if fiscal_year_start >= "2026-04-01":

    else:
        defense_tax = 0

### 14.4 J-KISS 転換 post-money cap の反復計算

```python
# post-money cap での転換株式数計算（反復解法）
def jkiss_conversion_post_money(
    jkiss_amount,        # J-KISS 払込金額
    post_money_cap,      # post-money valuation cap
    discount_rate,       # 例: 0.20
    series_a_price,      # Series A 発行価額（円/株）
    pre_money_shares,    # Series A 直前の発行済株式数（J-KISS 転換前）
    series_a_new_shares  # Series A 新規発行株式数
):
    discount_price = series_a_price * (1 - discount_rate)

    # post-money cap の場合、転換価額は反復が必要
    # cap_price = post_money_cap / (pre_money_shares + jkiss_converted_shares + series_a_new_shares)
    # jkiss_converted_shares = jkiss_amount / cap_price
    # 代数的に解く：
    # x = jkiss_converted_shares
    # post_money_cap / (pre + x + sa_new) = jkiss_amount / x
    # post_money_cap * x = jkiss_amount * (pre + x + sa_new)
    # post_money_cap * x - jkiss_amount * x = jkiss_amount * (pre + sa_new)
    # x * (post_money_cap - jkiss_amount) = jkiss_amount * (pre + sa_new)
    # x = jkiss_amount * (pre + sa_new) / (post_money_cap - jkiss_amount)

    cap_converted_shares = (
        jkiss_amount * (pre_money_shares + series_a_new_shares)
        / (post_money_cap - jkiss_amount)
    )
    cap_price = jkiss_amount / cap_converted_shares

    # MIN(discount, cap) で有利な方を選択
    if discount_price <= cap_price:
        # discount で転換
        converted_shares = jkiss_amount / discount_price
        conversion_price = discount_price
    else:
        # cap で転換
        converted_shares = cap_converted_shares
        conversion_price = cap_price

    return converted_shares, conversion_price
```

### 14.5 SaaS スタートアップ専用 KPI 行

日本の SaaS でも基本 KPI は米国と同じだが、消費税・インボイス・契約形態に固有の調整あり。

```
ARR（Annual Recurring Revenue、税抜ベース推奨）
MRR（Monthly Recurring Revenue）
契約 ACV（Annual Contract Value）
NRR（Net Revenue Retention）= (期首 ARR + Expansion - Downgrade - Churn) / 期首 ARR
GRR（Gross Revenue Retention）= (期首 ARR - Downgrade - Churn) / 期首 ARR

CAC（Customer Acquisition Cost）= 営業 + マーケ費用 / 新規顧客数
LTV（Lifetime Value）= ARPU × Gross Margin / Churn Rate
LTV/CAC 比 = LTV / CAC（目安：3 倍以上）
CAC Payback Period（月）= CAC / (ARPU × Gross Margin / 12)

Magic Number = (期間 ARR 増加額 × 4) / 前四半期の Sales & Marketing 費用
Rule of 40 = 売上成長率 (%) + EBITDA マージン (%)（40% 超が健全）
```

**日本特有の注意点**

- 契約は「自動更新型」が多く、年契約 + 月額前払いの組合せが標準。ARR 計算は月額 × 12 で一律ではなく、契約形態で調整。
- 消費税はモデル上「税抜」で統一推奨。インボイス制度導入後、課税事業者は明示的に税抜表示。
- 解約は「契約期間途中の解約不可」型と「いつでも解約可」型がある。後者は churn が即時反映される。
