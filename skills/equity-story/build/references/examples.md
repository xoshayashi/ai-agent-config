# Equity Story Examples

Use as compact patterns, not templates.

## Annotated Mini Example: ClinicFlow, US Series B

Thesis: ClinicFlow becomes the operating layer for specialty clinics because
claims-denial cost and nurse-admin shortages crossed in 2026.
> Annotation: start with the inflection, not the product.

Proof: 72% 90-day activation, 124% NRR, 74% gross margin, 8-month CAC payback,
and named vertical-SaaS comps sit beside the claims they prove.
> Annotation: numbers persuade only in context.

Market: bottom-up from 18k target clinics x workflow ACV, then triangulated
against claims-admin spend.
> Annotation: show capturable wedge before category size.

Bear case: integrations slow or payback extends; those observations falsify the
durability claim.
> Annotation: reversal triggers make the upside credible.

## Other Patterns

- Japan IPO: 結論 -> 市場環境 -> KPI -> 収益モデル -> 競争優位 -> リスク -> 想定問答。
- Diagnostic: map competitors by customer job, list 2-3 moat hypotheses, name
  falsification signals, then choose beachhead and what not to do.
