# Design Richness Rules

Use this file when a deck feels correct but flat. Richness is the ability to produce varied,
high-impact pages while preserving investment-bank / strategy-consulting discipline.

## Sources Of Richness

- Change focal object by slide: chart, table, value, diagram, image, quote, proof strip, or
  decision request.
- Change density intentionally: dense proof, quiet reset, current-location page, visual proof,
  or appendix.
- Use asymmetry with a clear alignment spine.
- Use scale contrast: one protagonist object large enough to carry the page.
- Use interpretation rails and evidence strips instead of generic card rows.
- Use color sparingly so one accent remains meaningful.

## Density Judgment

Increase density when:

- the reader needs comparison
- multiple evidence points verify one claim
- row/column structure makes the information easier to scan
- the page would otherwise feel underfilled

Reduce density when:

- two claims compete
- a chart/table becomes unreadable
- details belong in notes or appendix
- the page should function as a rhythm reset

Density must come with line spacing, role gaps, and clear hierarchy. Dense but cramped is a
failure.

## Rhythm

A deck should not be a sequence of identical proof pages. Alternate:

- overview and proof
- dense and quiet
- quantitative and structural
- comparison and implication
- argument and decision

Repeat grammar only when the reader is comparing similar things.

## Prohibited Freshness

- gradients, shadows, glass effects, neon colors, decorative icons
- card-heavy pages with no evidence reason
- large hero text unsupported by proof
- ornamental arrows
- closing slides that reuse a fixed end-card
- one-note color themes

## Special Layouts

Use unusual layout only when the evidence earns it. The page still needs a grid/flex contract,
source discipline, and a clear reading order.
