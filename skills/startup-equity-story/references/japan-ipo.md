# japan-ipo — 日本IPO・IR特化(成長可能性資料・KPI開示・引受審査・想定問答・継続開示・オファリング設計)

## 一本のストーリーへの接続(直前・直後の要素との「したがって」)

- 受け(直前要素=競争優位・成長戦略・財務計画から): 「なぜ勝てるか」と「どう伸びるか」が定まった。**したがって**日本IPOでは、その同じストーリーを東証の制度様式——「事業計画及び成長可能性に関する事項」(以下、成長可能性資料)の5大記載項目(ビジネスモデル/市場環境/競争力の源泉/事業計画/リスク情報)——に写像する。新しい話を作るのではなく、既存ストーリーの「制度翻訳」である。翻訳時に増えるのは3点のみ: (1)出典・算定方法つきの定量根拠、(2)IPOの目的(調達資金と成長戦略の紐付け=「なぜ上場しなければならないのか」)、(3)リスクと対応策。
- 送り(直後要素=上場後IR・資本市場との対話へ): 上場日にTDnetで開示した瞬間、ストーリーは「公約」に変わる。**したがって**上場後の決算説明・成長可能性資料の年次更新・中期経営計画は、すべて「上場時に示したストーリーの進捗報告」として設計する。上場時に語ったKPI・目標・資金使途を、上場後の資料が同じ定義・同じ粒度で追跡できること(トレーサビリティ)が、この要素の出口条件。

## ベスト構成

### 1. 成長可能性資料の章立て(制度項目→一本のストーリーへの変換)

東証の記載内容は「順序自由・統合可・様式自由(PDF、10MB上限、決算説明資料への統合も可)」。したがって制度項目を並べるのではなく、次の9章で一本の因果連鎖にする。

1. **投資ハイライト**(1-2p): 買う理由を3〜5点で先出し。各点は後章の見出しと1対1対応(Sansan方式: 本編はハイライト5点の証明に絞り、詳細・サービス仕様・PLはAppendixへ。同社は全66pの約5割がAppendix)。
2. **ミッション/解くべき課題**(1-2p): 顧客の構造的なペイン。市場章の「なぜ今か」の伏線。
3. **市場環境**(2-4p): ターゲット市場の内容(顧客種別・地域)と規模。第三者データ+出典が原則、独自推計は測定に用いたデータの出典と前提条件を詳細記載(留意事項の明文要求)。将来予測を書くなら予測の前提条件も。競合状況・自社ポジショニング・シェアを客観的事実で(ティーケーピー: 市場規模→自社シェア4%→獲得可能余地→他社比較表、が投資家の声で高評価)。
4. **サービス/プロダクト**(2-4p): 事業の流れ・仕入先/販売先の属性・進捗状況。複数事業は事業ごとに売上高・利益構成比と将来の寄与度変化。
5. **ビジネスモデル/収益構造**(2-3p): 収益・CF獲得の方法と主要費用構成。収益分配条件のある契約(プラットフォーム手数料等)は内容を開示。先行投資型は「将来の収益構造の変化見通し」が必須級。
6. **KPI実績(トラクション)**(3-5p): 下記2の3層設計で。
7. **競争力の源泉**(2-3p): 技術・知財・ノウハウ・ブランド・人材(経営陣)。「保有技術の有効性を示すデータ」等の客観的事実で裏付け。先行投資型は競争優位を「今後も獲得・維持する見込み」の具体記載が特に重要(留意事項)。
8. **成長戦略・事業計画**(4-6p): 競争力の源泉を「どう維持・強化するか」の観点で経営方針を書き、施策は研究開発・設備投資・マーケ・人員・資金計画の別に目標と達成見込み時期を付す。対象期間は各社設定+設定理由を記載。**新規上場日開示ではIPOの目的(調達資金をどの施策に充てるか)を明記**(トライアルHDは施策別・年度別の充当予定と実績を開示し高評価)。時価総額100億円未満は2030年基準への適合意識を明記。
9. **利益計画・経営指標+リスク情報**(2-4p): 中期利益計画は「記載可能な数値のみで足りるが、合理的な数値であること」。セグメント別計画値を推奨。リスクは有報「事業等のリスク」から成長関連を抜粋し、顕在化の可能性・時期・影響+**対応策を経営方針との関連で**記載。

分量目安: 本編25〜40p+Appendix。1章あたりの主張は1つ、各ページに定量根拠1つ以上。

### 2. KPI体系の3層設計(KPIは投資家との契約)

- 層1: PL/BS/CF計画(中期利益計画・セグメント別)。層2: その計画の根拠となるKPI(売上=顧客数×ARPU等のツリーに分解し、ツリーの各ノードに実績と目標を置く)。層3: そのKPIを重視する理由(ビジネスモデルとの接続)。
- 各KPIに必須の4点セット: **採用理由・実績値・目標値・算定方法**(独自指標は算定方法の開示が明文要求)。継続測定可能な指標を選ぶ(ユーザー数、単価、顧客獲得単価など)。
- 変更規律: KPIの追加・変更は理由を開示。主要KPIを変えても旧指標が合理的に算出可能なら継続記載する(東証留意事項)。「プライシング時だけ見せて上場後に引っ込める」は機関投資家が名指しで不誠実と評価(東証・投資家の期待#7: 「一度示したKPIの進捗は継続して開示すべき」)。
- 見せ方の到達点: GA technologies型——中計の売上・利益目標→事業別ブレークダウン→事業別の重要KPI(市場シェア・件数・単価)→進捗率バー表示。M&A活用企業は案件別の期待シナジーと実施後進捗も(ティーケーピー)。

### 3. 引受審査・想定問答

- 引受審査(日証協「有価証券の引受け等に関する規則」16条+細則): 公開適格性/企業経営の健全性・独立性/事業継続体制/コーポレート・ガバナンスと内部管理体制/業績見通し(事業計画の合理性)/調達する資金の使途/情報開示の適切性——が審査項目。エクイティストーリー側の含意: **成長可能性資料に書く数値は引受審査で説明した事業計画と同一線上**に置く(審査で通らない数字を開示資料に書かない、逆も然り)。
- 想定問答集の作り方: 主幹事・弁護士とすり合わせ、目論見書(有価証券届出書)+成長可能性資料の記載範囲を「回答可能領域」として定義する。フェア・ディスクロージャー・ルール(金商法27条の36)により、個別面談で未公表の重要情報(業績の確定的情報等)を伝えれば公表義務が発生する。**回答は「開示済み情報の再構成+ロジックの深掘り」に限定**し、未開示の月次数値・未公表の翌期見通しは答えない設計にする。
- 質問カテゴリ(最低この10領域を各3問): (1)市場規模の算定根拠と前提、(2)競合と勝ち筋・シェアの持続性、(3)KPIの定義とドライバー(単価/チャーン/獲得効率)、(4)利益率の中期パス(先行投資の回収時期)、(5)資金使途とマイルストーン、(6)経営陣・キーパーソン依存、(7)大株主構成・ロックアップ・VC売却意向、(8)規制・許認可リスク、(9)M&A方針と規律、(10)資本政策・還元方針(成長期は「配当より成長投資」を明言——投資家の期待#3)。
- 投資家タイプ別傾向: 中長期ロングは市場規模の上限・競争優位の持続性・経営者の資本配分思想を、短期筋はガイダンスの保守度・四半期の谷・需給(ロックアップ解除・売出し圧力)を突く。ロング向けには「時価総額100億円は通過点」の先のシナリオまで用意する(投資家の期待#1)。

### 4. オファリング設計

- 公募(新株発行)を軸にする。売出しのみ・売出し偏重は「成長資金が要らない=なぜ上場するのか」への回答不能を意味し、機関投資家の不信を招く。増資への正しい反論は「アクリーティブな増資なら株価は下がらない。問題は成長への道筋を示せていないこと」(東証・投資家の期待#2)。調達額は成長戦略の施策別資金計画から積み上げ、使途と時間軸を明示する。
- 制度知識(2022年公取委報告→日証協改革): オーバーアロットメントは上限15%で明確化、親引けガイドライン改正、2023年10月以降は**仮条件レンジ外での公開価格設定・売出株式数の柔軟な変更・上場日程の短縮/柔軟化**が可能。ロードショーのフィードバックを価格・株数に反映できる前提で needs 把握を設計する。
- 想定時価総額の目安: 機関投資家の中長期投資対象は時価総額100億円が一つの目安。未満で上場する場合は「まず個人投資家をメインターゲットに流動性の土壌を作り、段階的に機関投資家を取り込む」IR設計を上場時から織り込む(投資家の期待#6)。

### 5. 上場後IRへの接続(継続開示・ガイダンス)

- 更新義務: 成長可能性資料は少なくとも1事業年度に1回以上(うち1回は事業年度経過後3か月以内)、事業計画の見直しや重要な変更時は速やかに更新。進捗状況の節では「前回記載事項の達成状況」「IPOの目的が実現できているか」「更新がない事項はその旨」「記載取りやめは理由」を書く。次回開示予定時期も記載。
- ガイダンス設計(meet-beat-raise): 期初予想は達成確度が高い水準(実務目安80〜90%)に置き、上振れ→上方修正のリズムを作る。ただし極端な保守化はガイダンスリスク(市場期待未達による失望売り)を生むため、アナリストコンセンサスとの距離を主幹事・IRで確認して設定する。
- 未達時: 下方修正自体より「要因分析と改善施策が示されないこと」が信頼を壊す(投資家の期待#7)。ティーケーピー型の開示——当初計画・実績・差異・背景・見直し後計画を1表で——を標準とする。赤字先行投資型は「投資の狙い・回収時期・投資継続/撤退の判断基準」を事前に開示しておくと未達時の説明が成立する(アストロスケール、キャンバスの事例)。
- グロース市場改革の影響: 上場維持基準が「10年経過後40億円」→**「5年経過後100億円」(2030年3月1日以後適用)**に厳格化。エクイティストーリーは「上場5年で時価総額100億円に到達する成長率(例: 年20〜30%成長の継続で3〜5年で倍増)」を逆算して市場規模・戦略・KPI目標を張る必要がある。時価総額100億円未満の上場会社は成長可能性資料に新基準適合への施策記載を要請済み。M&A・提携などインオーガニック戦略や「他社に買われる」選択肢も再構築の論点(投資家の期待#5)。

## 合格基準チェックリスト(検証可能な形)

- [ ] 投資ハイライト3〜5点があり、各点が本編の章と1対1対応している
- [ ] 5大記載項目(ビジネスモデル/市場環境/競争力の源泉/事業計画/リスク情報)がすべて回収されている
- [ ] 市場規模に出典がある。独自推計は元データの出典+前提条件が明記されている
- [ ] 競合比較が客観的事実(第三者データ・公開情報)に基づき、自社ポジショニング/シェアが示されている
- [ ] 主要KPIすべてに採用理由・実績値・目標値・算定方法の4点が揃っている
- [ ] PL計画→根拠KPI→重視理由の3層が接続し、KPIツリーで売上・利益が分解されている
- [ ] 複数事業の場合、事業別の売上・利益構成比と将来の寄与度変化が書かれている
- [ ] 成長戦略の施策に研究開発/設備投資/マーケ/人員/資金の計画と達成見込み時期が付いている
- [ ] IPOの目的(調達資金と施策の紐付け)が金額レベルで書かれている(新規上場時)/使途実績が更新されている(上場後)
- [ ] 中期利益計画の数値が引受審査に耐える「合理的な数値」で、前提条件が明記されている
- [ ] リスクごとに顕在化の可能性・時期・影響+対応策(経営方針との関連)がある
- [ ] 先行投資型: 赤字の要因=投資である説明、回収時期、投資継続/変更の判断基準がある
- [ ] 想定問答が10カテゴリ×3問以上あり、全回答が開示済み情報の範囲内(FDルール適合)である
- [ ] オファリングは公募中心で、調達額が資金計画から積み上がっている
- [ ] 上場5年後時価総額100億円への到達シナリオが成長率として逆算されている
- [ ] 上場後資料が上場時と同じKPI定義・粒度で予実(計画・実績・差異・背景)を追跡できる

## 致命的失敗パターン

1. **プライシング時限定KPI**: 上場時だけ有利なKPIを見せ、上場後に開示をやめる。東証が公表した機関投資家意見で「不誠実」と名指しされる典型。
2. **売出しのみ・売出し偏重IPO**: 「なぜ上場しなければならないのか」に答えられず、上場ゴールと解釈される。
3. **出典なきTAM・ホッケースティック計画**: 独自推計の前提非開示、過去実績と非連続な急成長カーブに根拠がない。引受審査(業績見通しの妥当性)でも差し戻される。
4. **制度項目の逐条埋め**: 5項目を順番に埋めただけで因果がなく、ハイライトなし・各章の主張なし。読後に「なぜこの会社が勝つのか」が言えない。
5. **未達の黙殺**: 下方修正や KPI 未達時に要因分析・改善策を示さない、進捗記載を無言で削る(取りやめは理由開示が義務)。
6. **赤字の説明放棄**: 先行投資型なのに投資の狙い・回収時期を語らず、「赤字では評価されない」と成長投資自体を萎縮させる(投資家は道筋があれば赤字でも買う——投資家の期待#4)。
7. **FDルール違反型IR**: ロードショー・個別面談で未公表の確定的な業績情報を一部投資家にだけ伝える。公表義務発生+信頼毀損。
8. **上場時と上場後の断絶**: 決算説明資料が成長可能性資料と別のKPI・別のセグメント区分で作られ、公約の検証が不能になる。

## ステージ差

- **Seed**: 制度対応は不要。ただし将来のKPI契約を見据え、KPIの定義・算定方法を最初から文書化し変更履歴を残す(後で「算定方法変更」を繰り返す会社は信頼を失う)。市場規模推計は出典つきで作る習慣をここで確立。
- **Series A〜Growth(N-3〜N-2期)**: VC向けピッチを成長可能性資料の9章構成に「先行翻訳」してギャップを洗う——出典なきTAM、算定方法が揺れるKPI、資金使途の粒度不足が典型ギャップ。N-3期から月次予実管理を回し、予実差異±10%以内の着地精度を実績として積む(引受審査の業績見通し評価は予算統制の実績で見られる)。主幹事選定時にはエクイティストーリーのドラフトを自社で持参し、審査で説明する事業計画と対外ストーリーを最初から一本化する。上場後5年で時価総額100億円の逆算から、上場タイミング自体(もう1ラウンド調達して規模を作ってから上場する選択肢)を検討する。
- **IPO・IR(本役)**: 本文の全節を適用。上場承認〜上場日はロードショー(想定問答・FD制約下の回答設計)、上場日に成長可能性資料を開示、以後年1回以上更新+決算説明会・個別面談の実施状況をCG報告書「IRに関する活動状況」に記載(参加投資家の属性、動画・書き起こしの公開状況まで)。時価総額100億円未満の間は個人投資家向けIR(説明会定期開催、Q&A公開、平易な資料)で流動性を作り、機関投資家カバレッジへ段階移行する。
